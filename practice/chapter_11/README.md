6、在重载运算符 =、()、[]和 ->时，有什么限制？

=​
必须是非静态成员函数​
1. 注意自赋值检查
2. 通常返回 ClassName&
3. 用于深拷贝

()​
必须是非静态成员函数​
1. 参数和返回值灵活
2. 创建“函数对象”

[]​
必须是非静态成员函数​
1. 通常提供常量和非常量两个版本
2. 非常量版本返回引用

->​
必须是非静态成员函数​
1. 必须返回一个指针，或一个重载了 ->的对象
2. 有特殊的“传递”语义
